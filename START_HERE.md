# Start here

1. Open `headroom.code-workspace` in Cursor.
2. Let Cursor read `AGENTS.md` and `.cursor/rules/headroom.mdc` before making changes.
3. Run the application in deterministic demo mode first.
4. Validate `/dashboard`, `/record/headroom?scenario=critical`, and `/guide`.
5. Add only one read-only provider integration at a time.
6. Never place real credentials or exact private usage figures in screenshots.
